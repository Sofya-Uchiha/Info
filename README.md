# Работа с git
### Скачать проект:
``` bash
git clone <URL>
```
### Учесть все файлы:
``` bash
git add .
```
### Сохранить коммит:
``` bash
git commit -m "..."
```
### Отправить в облако:
``` bash
git push
```

### Отправка в облако:
``` bash
git push
```
### Принять изменения в облако:
``` bash
git pull
```
